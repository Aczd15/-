-Выберите действие: 
 - добавить кандидата 
 - удалить кандидата
 - просмотреть всех кандидатов
 - выход

1. Добавить кандидата - Этот раздел описывает, как добавить нового кандидата в нашу систему. Пожалуйста, следуйте инструкциям внимательно.


 - После перехода на строку добавления кандидата вы увидите подсказки для правильного заполнения данных кандидатов, где нужно заполнить необходимую информацию. Поля могут различаться в зависимости от системы, но обычно включают в себя:


 - Введите пол (М/Ж):
 - Введите регистрационный номер:
 - Введите дату регистрации (Год-Месяц-Число):
 - Введите сведения о себе:
 - Введите требования к партнеру:
 

 - После добавления кандидата проверьте успешно ли добавлена анкета в базу.
 - При необходимости вы можете удалить свою анкету, и пересоздать её заного.

2. Удалить кандидата - этот пункт описывает, как удалить данные кандидата из системы, Удаление кандидата является необратимым действием и происходит по его регистрационному номеру, после удаления вам придется заного создать свою анкету 


- Перейдите в раздел просмотреть всех кандидатов и найдите нужного кандидата и регистрационный номер.


- После нахождения нужного кандидата перейдите в раздел удалить кандидата и введите его регистрациооный номер который вы нашли немного ранее и нажмите enter после этого действия выбранная анкета удалиться без функции восстановления. 

3. Просмотреть всех кандидатов - Эта функция предназначена для просмотра всех кандидатов в нашей базе.


4. Выход - эта функиця предназначена для выхода из системы.
